# Demo_Github_Actions_CI


## Installation

Créer un environnement virtuel :

python -m venv venv


Activer :

Windows :

venv\Scripts\activate


Installer les dépendances :

make install


## Tests

Lancer les tests :

make test


## Lint

Vérifier le code :

make lint


## Exécution

Lancer le programme :

make run
